# Programación 3 (Arquitecturas con IA)

*Plan 2026 — Analista Programador — SEMESTRE 3*

---

## OBJETIVOS

- Profundizar en el paradigma de POO con énfasis en arquitecturas empresariales y soporte de bases de datos relacionales.
- Implementar soluciones robustas utilizando patrones de diseño y arquitecturas de capas.
- Dominar el uso de IA como motor de productividad para la generación de código técnico avanzado.

## CONTENIDOS DETALLADOS

- **Arquitectura en Capas:** Separación de lógica de presentación, negocio (dominio) y acceso a datos.
- **Manejo de Persistencia:** Uso de Object Relational Mapping (ORM) mediante Entity Framework.
- **ASP.NET MVC Avanzado:** Sistema de enrutamiento avanzado, filtros, Web API y servicios REST.
- **Patrones de Diseño:** Implementación de Repository, Unit of Work y Dependency Injection.
- **IA Obligatoria como Copilot:**
  - Generación obligatoria de Pruebas Unitarias asistidas por IA para asegurar la calidad del software.
  - Refactorización automática de código para mejorar la legibilidad y el rendimiento.
  - Uso de IA para la generación de documentación técnica bajo estándares de industria.

## APROBACIÓN

- **Régimen:** Materia exonerable con 86 puntos.
- **Requisitos:** Ganancia de curso con 70 puntos mediante trabajos obligatorios y parciales.
- **Evaluación:** Defensa de una aplicación Full-Stack que integre APIs de terceros y persistencia compleja.

## CARGA HORARIA

96 horas frontales (6 horas semanales).
